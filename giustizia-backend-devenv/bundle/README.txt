Al primo startup è necessario creare un dominio per weblogic. E' sufficiente digitare il comando:

/home/developer/wls12210/oracle_common/common/bin/config.sh

NOTA: è definito un volume ben preciso che è sul path: /home/developer/wls12210/user_projects
quindi è fortemente consigliato usare questo path. In questo modo le operazioni di weblogic saranno più veloci e i domini portabili.


Per lanciare eclipse è sufficiente digitare il comando da command line.

NOTA: è definito un volume ben preciso che è sul path: /home/developer/workspaces
quindi è fortemente consigliato usare questo path. In questo eclipse sarà più veloce e i workspaces portabili.

Per configurare correttamente eclipse è necessario installare il plugin weblogic tools (quello di JBoss è preinstallato)
